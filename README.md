# lantern 🔦

> Illuminate your beacon node. Live.

Lantern is a local debug dashboard for Ethereum beacon node operators.
Connect it to any beacon node (Prysm, Lighthouse, Teku, Nimbus) and see
fork choice updates, reorgs, attestation delays, and missed slots — live,
in your browser, with no external dependencies.
